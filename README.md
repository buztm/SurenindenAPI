# Süreninden Araç Kiralama Portalı

Bu github repository **İnternet Programcılığı II** _Finali_ kapsamında hazırlanmıştır.
Final konusu Araç Kiralama Portalı olmaktadır.

## Kullanılan Teknolojiler
Projeyi geliştirmek için 
* **Visual Studio 2022** ide kullanılmıştır.
* .Net 9.0 Core Web API ve MVC projesidir.
* SQL Server veri tabanı ile Code-First.

FrontEnd Tarafında
* Jquery AJAX
* Bootstrap 5 kullanılmıştır.
